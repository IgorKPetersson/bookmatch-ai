MVP (Minimum Viable Product)

Detta räcker för VG:
1. User system
- Register
- Login
- Får automatiskt ett bibliotek

2. CRUD (krav)
- Skapa bok i "Mitt bibliotek"
- Se sina böcker
- Uppdatera status (läst / vill läsa)
- Ta bort bok

3️. Rekommendationer
- User anger 3 böcker
- Backend skickar till AI-modul

Får tillbaka:
- 3–5 rekommendationer
- Förklaring per bok
- Kan spara rekommendation i sitt bibliotek

Det matchar exakt er pitch + PDF-arkitektur
